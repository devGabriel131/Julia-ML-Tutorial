# Notebooks — your work goes here

Drop every Pluto notebook you work in (`.jl`) into this folder and push.
Each daily editor run pulls this repo first, reads your notebooks cell by cell,
grades them against the gradebook, and aims the next book improvement at what
they reveal.

Suggested naming: `M1_practice.jl`, `M1_exam_attempt.jl`, `M2_checkpoints.jl`, …
